How to Open with Live Server

  1. Save this file with an .html extension (e.g., jewelry-store.html)
  2. Open Visual Studio Code
  3. Install the Live Server extension if you haven't already:
    • Go to Extensions (Ctrl+Shift+X)
    • Search for "Live Server"
    • Click Install
  4. Right-click on the HTML file in VS Code
  5. Select "Open with Live Server"
  6. Your website will open in your default browser at http://127.0.0.1:5500/
